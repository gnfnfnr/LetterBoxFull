4. `./gradlew.bat build`: 프로젝트를 빌드한다

5. `cd .\build\libs\`: 서버 파일로 이동한다
6. `java -jar .\LetterBox-0.0.1-SNAPSHOT.jar`: 서버를 킨다(백을 키는 과정)
